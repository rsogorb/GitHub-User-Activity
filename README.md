# GitHub-User-Activity
Uses GitHub API to fetch user activity and display it in the terminal.

Solution for the following project: https://roadmap.sh/projects/github-user-activity

## How to Run
1. **Clona el repositorio**:

```bash
git clone <URL_DEL_REPOSITORIO>
cd <NOMBRE_DEL_REPOSITORIO>
```
2.  Ejecuta la aplicación en la terminal pasando el nombre de usuario de GitHub como argumento:

```bash
node index.js <github-username>
